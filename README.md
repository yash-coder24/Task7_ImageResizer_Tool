# Task7_ImageResizer_Tool

# Objective
Resize and convert multiple images in a folder automatically.

# Tools
Python,
Pillow (PIL),
os module.

# Project Structure
image_resizer/
│
├── input_images/      # original images
├── output_images/     # resized images
└── image_resizer.py

# Features
Batch resize images,
Supports JPG, PNG, JPEG,jfif,
Auto-create output folder,
Optional format conversion.

# Concepts Used
os.listdir(),
os.path.join(),
File handling,
Exception handling,
Pillow Image processing.

# Image Resizer Tool
Developed a batch image processing tool using Python and Pillow to resize and convert images automatically. Implemented folder-based processing, format validation, and exception handling for robust automation.
